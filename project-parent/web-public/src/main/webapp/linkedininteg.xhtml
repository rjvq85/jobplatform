Integração com LinkedIn

A nossa plataforma permite a integração segura com o LinkedIn, permitindo desta forma recolher mais informações sobre o seu perfil profissional.

Apenas precisa de seguir os passos apresentados.

Primeiro passo

Clicar no botão

Irá abrir uma nova janela. Garanta, por favor, que o seu 'browser' não está a bloquear novos pop-ups

Segundo passo

Introduzir as suas credenciais do LinkedIn na nova página

Se o fizer com sucesso, ser-lhe-á fornecido um número. Guarde-o!

Terceiro passo

Introduzir o número fornecido

Iremos guardar alguns dados como a sua fotografia, ligação para o seu perfil, etc...
